## Overview

The scraper fetches menu data from the university's ASP-based website, parses HTML, converts encoding, and saves JSON files.

## Quick start

Run the scraper from the project root:

```bash
pnpm scrape
```

This executes `scripts/scrape-menu.ts` and outputs to `public/data/<YYYY-MM>/menu-YYYYMMDD.json`.

## How it works

### 1. Fetch HTML

The scraper fetches the university's menu page:

```typescript
const response = await fetch('https://yemekhane.cu.edu.tr/default.asp')
const buffer = await response.arrayBuffer()
```

### 2. Convert encoding

The website uses **Windows-1254** encoding (Turkish). We convert to UTF-8 using `iconv-lite`:

```typescript
import iconv from 'iconv-lite'

const html = iconv.decode(Buffer.from(buffer), 'windows-1254')
```

### 3. Parse with Cheerio

Cheerio provides jQuery-like DOM traversal:

```typescript
import * as cheerio from 'cheerio'

const $ = cheerio.load(html)
const dayCards = $('.col-md-2, .col-md-3')  // Day cards
```

### 4. Extract menu data

For each day card:

<Accordion title="Extract date and YMK">
  ```typescript
  const dateLink = $card.find('FONT[size="6"] a').attr('href')
  const ymkMatch = dateLink?.match(/ymk=(\d+)/)
  const ymk = ymkMatch ? parseInt(ymkMatch[1]) : null
  
  const dateText = $card.find('FONT[size="6"]').text()
  const dateMatch = dateText.match(/(\d{1,2})\.(\d{2})\.(\d{4})/)
  const [, day, month, year] = dateMatch
  const date = `${year}-${month}-${day.padStart(2, '0')}`  // "2026-02-19"
  ```
</Accordion>

<Accordion title="Extract meals">
  ```typescript
  $card.find('ul li a').each((i, el) => {
    const $link = $(el)
    const href = $link.attr('href') || ''
    
    // Extract ID: yemek-goster.asp?id=157 → "157"
    const idMatch = href.match(/id=(\d+)/)
    const id = idMatch[1]
    
    // Parse name and calories from link HTML
    const linkHtml = $link.html() || ''
    const parts = linkHtml.split('<br>')
    const name = parts[0]?.trim()  // "Ekşili Köfte"
    const caloriesText = parts[1] || ''  // "294 Kalori"
    const caloriesMatch = caloriesText.match(/(\d+)/)
    const calories = caloriesMatch ? parseInt(caloriesMatch[1]) : 0
    
    meals.push({ id, name, calories, category: 'ana_yemek' })
  })
  ```
</Accordion>

<Accordion title="Extract total calories">
  ```typescript
  const calorieSpans = $card.find('span[style*="color: rgb(204, 0, 0)"]')
  calorieSpans.each((i, el) => {
    const text = $(el).text()
    // "Toplam 797 Kalori" or "Ana Yemekli : 891 Kalori"
    const match = text.match(/(\d+)/)
    if (match) {
      const cal = parseInt(match[1])
      if (text.includes('Ana Yemekli') || i === 0) {
        totalCalories = cal
      }
    }
  })
  ```
</Accordion>

### 5. Save to JSON

The scraper saves data to `public/data/<YYYY-MM>/menu-YYYYMMDD.json`:

```typescript
const monthDir = path.join(__dirname, '..', 'public', 'data', menuData.month)
fs.mkdirSync(monthDir, { recursive: true })

const filename = `menu-${scrapeDateCompact}.json`  // menu-20260219.json
const filePath = path.join(monthDir, filename)

fs.writeFileSync(filePath, JSON.stringify(menuData, null, 2), 'utf-8')
```

### 6. Cleanup old files

Keeps only the 5 most recent JSON files per month:

```typescript
const files = fs.readdirSync(monthDir)
  .filter(file => file.endsWith('.json') && file.startsWith('menu-'))
  .map(file => ({ name: file, path: path.join(monthDir, file) }))
  .sort((a, b) => b.name.localeCompare(a.name))  // Newest first

if (files.length > 5) {
  files.slice(5).forEach(file => fs.unlinkSync(file.path))
}
```

## TypeScript types

Defined in `lib/types.ts`:

```typescript
export interface MenuData {
  month: string          // "2026-02"
  lastUpdated: string    // ISO date
  scrapeDate: string     // "2026-02-19"
  totalDays: number
  days: DayMenu[]
}

export interface DayMenu {
  ymk: number           // University's internal ID
  date: string          // "2026-02-19"
  dayName: string       // "Çarşamba"
  hasData: boolean      // false if no menu data
  totalCalories: number
  meals: Meal[]
}

export interface Meal {
  id: string            // "157"
  name: string          // "Ekşili Köfte"
  calories: number      // 294
  category: MealCategory
}

export type MealCategory =
  | "ana_yemek"
  | "yan_yemek"
  | "corba"
  | "yan_urun"
  | "tatli"
  | "icecek"
```

## Automated scraping

GitHub Actions runs the scraper daily at 07:00 (UTC+3).

**Workflow**: `.github/workflows/menu-update.yml`

```yaml
name: Update Menu Data

on:
  schedule:
    - cron: '0 4 * * *'  # 07:00 Turkey time (UTC+3)
  workflow_dispatch:     # Manual trigger

jobs:
  scrape:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '18'
      - run: npm install
      - run: npm run scrape
      - name: Commit and push
        run: |
          git config user.name "GitHub Actions"
          git config user.email "actions@github.com"
          git add public/data/
          git commit -m "Update menu data" || exit 0
          git push
```

## Modifying the scraper

### Change scraping logic

Edit `lib/scraper.ts`:

```typescript

interface DayMenu {
  ymk: number
  date: string
  dayName: string
  hasData: boolean
  totalCalories: number
  meals: Meal[]
  specialNote?: string  // New field
}

// Extract from HTML
const specialNote = $card.find('.special-note').text()
days.push({ ...existingFields, specialNote })
```

### Add meal categories

Currently all meals are categorized as `ana_yemek`. To categorize by position:

```typescript
const categories: MealCategory[] = ['corba', 'ana_yemek', 'yan_yemek', 'tatli', 'icecek']

$card.find('ul li a').each((i, el) => {
  // ... existing parsing logic
  
  const category = categories[i] || 'ana_yemek'
  meals.push({ id, name, calories, category })
})
```

### Change output directory

Edit `scripts/scrape-menu.ts`:

```typescript
const dataDir = path.join(__dirname, '..', 'public', 'menu-data')  // Changed
const monthDir = path.join(dataDir, menuData.month)
```

## Error handling

<Accordion title="HTTP error">
  ```typescript
  if (!response.ok) {
    throw new Error(`HTTP error! status: ${response.status}`)
  }
  ```
  
  **Solution**: Check if the university website is accessible.
</Accordion>

<Accordion title="Encoding error">
  ```typescript
  const html = iconv.decode(Buffer.from(buffer), 'windows-1254')
  ```
  
  **Solution**: Verify the website still uses Windows-1254 encoding. Try other encodings if needed.
</Accordion>

<Accordion title="Parse error">
  ```typescript
  const dayCards = $('.col-md-2, .col-md-3')
  if (dayCards.length === 0) {
    console.warn('No day cards found')
  }
  ```
  
  **Solution**: The website HTML structure may have changed. Inspect the page and update selectors.
</Accordion>

## Scraping best practices

1. **Respect rate limits**: Don't scrape too frequently (daily is sufficient)
2. **Handle errors gracefully**: Log errors, don't crash the script
3. **Validate data**: Check for required fields before saving
4. **Keep backups**: GitHub keeps history of JSON files
5. **Monitor changes**: Set up alerts if scraping fails
